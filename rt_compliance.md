## 1. 검사 범위 및 근거

IGF Code Ch.16 및 LR 선급 규칙상 가스연료 배관의 맞대기 용접부는 설계압력/설치
구역 조건에 따라 **100% 체적검사(RT 또는 UT)** 대상이다. 분기 아웃렛은 내관에서
**냉간성형(extruded collar)으로 일체 성형**되어 분기 연결부 자체는 무용접이며,
체적검사 대상 용접부는 **아웃렛–플랜지 거스 용접 1개소**뿐이다. 본 용접부는
이중관 환형부 내 협소 구역에 위치하며, 곡률·구경(OD 26.7) 특성상 UT 적용이
곤란하므로 RT(DWDI)를 선정하였다.

## 2. 기법 요약 (Technique Data)

## 3. 적합성 매트릭스

### 3.1 LR 선급 규칙 (Lloyd's Register)

| 요구사항 | 근거 | 본 세팅 | 판정 |
|---|---|---|---|
| 가스연료 배관 맞대기 용접 100% 체적검사 | LR Rules (IGF Code Ch.16.4 이행) | 전둘레 3회 촬영, 360° 커버 | **적합** |
| RT 기법은 ISO 17636-1 또는 동등 규격 준수 | LR Rules for Materials, Ch.13 (NDE) | ISO 17636-1 기반 기법서 (아래 3.2) | **적합** |
| 용접 품질등급 ISO 5817 Level B | LR 신조 용접 요구 | 판정기준 ISO 10675-1 Acceptance Level 1 적용 | **적합** |
| NDE 요원 자격: ISO 9712 또는 동등(SNT-TC-1A 등) Level 2 이상 | LR Rules for Materials, Ch.13 | 검사·판독자 Level 2 지정 (시행 시 확인) | **적합(조건)** |
| 검사 절차서 사전 승인 | LR 검사관 입회/승인 관행 | 본 문서 + 기법서 제출 대상 | 시행 시 확인 |

### 3.2 ISO 17636-1 (필름 RT 기법)

| 요구사항 | 조항 | 요구치 | 본 세팅 | 판정 |
|---|---|---|---|---|
| 기법 등급 | §5 | Class A 또는 B | Class A 충족 (B는 §비고 1 참조) | **적합** |
| 최소 SOD (Class A) | §7.6, 식 f/d ≥ 7.5·b^(2/3) | SOD ≥ 167 mm | 173 mm | **적합** |
| DWDI 중첩기법 최소 촬영수 | §7.7 / Annex A | 60° 또는 120° 간격 3회 | 3회 @ 60° | **적합** |
| 타원/중첩 기법 선택 | Annex A | t/OD 및 분리 가능성 기준 | 고도각 제한으로 중첩기법 채택 | **적합** |
| IQI 종류/배치 | §7.8, ISO 19232-1 | 와이어형, 선원측 원칙, 용접선 직각 | 와이어형 W10–W16, 선원측 래핑 | **적합** |
| 요구 식별 와이어 (Class A, w≈5.7 mm) | Table B.1 | W14 (Ø0.16 mm) 이상 | 모의 필름상 W13–14 식별 수준, 시행 시 실측 확인 | **적합(조건)** |
| 필름 농도 | §7.9 | ≥ 2.0 (Class A) | 노출 조건으로 달성 (시행 시 농도계 확인) | **적합(조건)** |
| 감마선원 적용 두께 범위 | §7.1.2, Table 2 | Ir-192 권장범위 미만 (w≈5.7 < 10 mm) | 동 조항 NOTE: 접근 제한 시 합의 하에 적용 가능, IQI로 감도 입증 | **적합(합의 필요)** |

> **비고 1 (Class B 대응)**: Class B 요구 시 SOD ≥ 333 mm. 선원측은 개방 공간이므로
> SFD를 ≥ 370 mm로 연장(U_g ≈ 0.22 mm)하거나 소초점(f=1.0 mm) Ir-192/Se-75
> 선원으로 교체하면 현 배치 그대로 충족 가능 (`rt_calculation.md` 참조).

## 4. 판정 기준

- 용접 품질등급: **ISO 5817 Level B** (신조 선박 가스연료 배관)
- RT 판정등급: **ISO 10675-1 Acceptance Level 1**
- 불합격 지시 발생 시: 보수 후 동일 기법 재촬영

## 5. 결론

본 RT 세팅은 **LR 규칙(ISO 17636-1 Class A 기준), ASME Sec. V / ASNT 요구사항을
충족**한다. 다음 3개 항목은 시행 단계에서 입증·확인을 요한다.

1. 필름상 요구 IQI 와이어(W14) 식별 및 농도(≥2.0) 실측 확인
2. 검사 요원 자격 증빙 (ISO 9712 / SNT-TC-1A Level 2)
3. 박육(w<10 mm)에 대한 Ir-192 적용 합의 (ISO 17636-1 §7.1.2 NOTE에 따른
   선급 검사관 합의) — 또는 Se-75 소초점 선원 채택으로 동시 해소 가능

Class B 기법이 지정되는 경우 비고 1의 SFD 연장(≥345 mm)만으로 충족된다.
