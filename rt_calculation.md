# 분기 거스 용접부 RT 계산서

대상: IGF 이중관 분기 아웃렛–WN 플랜지 거스 용접부 (넥 OD 26.7 / t 2.87, Ir-192, DWDI)

## 촬영 배치 (viewer.html `?rt=1` 시각화와 동일)

| 항목 | 값 | 비고 |
|---|---|---|
| 기법 | DWDI (이중벽 양면상) | 고도각 ≈6° → 사실상 중첩 기법에 가까움 |
| 촬영 횟수 | 3회 | 방위 -60° / 0° / +60° (60° 간격) |
| 선원 위치 | 분기축에서 수평 185 mm, 용접면 +20 mm | 컷아웃 개구부(±z) 섹터에서만 접근 가능 |
| 필름 | R24 곡면 스트립(스탠드오프 4 mm), 호 103°, 선원 반대쪽 | 용접부 비밀착 — 하단은 내관 표면, 상단은 플랜지 디스크 하면에 테이프 고정 |
| IQI | ISO 19232 와이어형, 선원측 용접비드 위 래핑 | 와이어는 용접선과 수직 |
| 커버리지 | 필름쪽 상 + 선원쪽 상 합산으로 360° 전둘레 | 60° 간격 3회로 빠짐없음 |

## 기하 계산

- SFD (선원→필름) = sqrt(209² + 20²) ≈ **210 mm**
- SOD (선원→선원측 용접 표면) ≈ **173 mm**
- b (피사체→필름) = SFD − SOD ≈ **37 mm** (선원측 벽 기준, 최악값 — 필름 스탠드오프 4 mm 포함)
- 확대율 M = SFD / SOD ≈ **1.22**

## 기하학적 불선명도 Ug

Ug = f · b / SOD = 2.0 × 37 / 173 ≈ **0.43 mm**  (f = Ir-192 초점 2.0 mm 가정)

| 기준 | 요구치 | 현재 배치 | 판정 |
|---|---|---|---|
| ASME Sec. V T-274.2 (t < 50.8 mm) | Ug ≤ 0.51 mm | 0.43 mm | 합격 |
| ISO 17636-1 Class A: SOD ≥ 7.5·f·b^(2/3) | ≥ 167 mm | 173 mm | 합격 (여유 6 mm) |
| ISO 17636-1 Class B: SOD ≥ 15·f·b^(2/3) | ≥ 333 mm | 173 mm | 불합격 |

> 필름 스탠드오프 비고: 규격 요구는 "가능한 한 필름을 검사체에 밀착"이며 밀착
> 자체가 의무는 아니다. 용접선이 내관 표면 바로 위에 있어 밀착 시 스트립이
> 좌굴되므로, R24 스탠드오프 + 테이프 고정 배치를 채택하고 그에 따른 b 증가를
> 상기 Ug/최소거리 계산에 반영하였다 (선원 반경 170→185 mm로 연장하여 보상).

## Class B(선급 요구 시) 충족 방안

1. **SFD 연장**: 선원은 컷아웃 바깥 개방 공간에 있으므로 제약 없음.
   SFD ≈ 370 mm → Ug ≈ 0.22 mm (합격). 노출시간 약 3.1배 증가.
2. **소초점 선원**: f = 1.0 mm급 Ir-192 또는 Se-75 사용 시 현재 거리 그대로 합격
   (Ug ≈ 0.21 mm). Se-75는 박육(2.87 mm)에서 콘트라스트도 유리.
